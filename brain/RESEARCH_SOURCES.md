# Research Sources — Lean Stack
*Token rule: SEARCH, don't FETCH. A WebSearch returns cheap snippets across many*
*sources at once. A WebFetch downloads a whole page (10-40k tokens). Default to*
*search. Fetch a page only when a specific number/fact you need is missing from*
*every snippet — and never more than 1-2 fetches in a run.*

---

## THE 12 CORE SOURCES (pick 3-5 per run, by today's targets)

Don't visit these. SEARCH them with `site:` filters so one query mines the source.

### Demand (what people want and pay for)
1. **kariera.gr** — jobs = what's scaling. `site:kariera.gr "[role]"`
2. **skroutz.gr** — real prices + demand. `site:skroutz.gr [product]`
3. **Google Trends GR** — `[category] google trends Greece` (search the snippet, don't open)

### What changed (24-48h news scan — one query each, max)
4. **ekathimerini.com/economy** — `site:ekathimerini.com [topic] [this month]`
5. **naftemporiki.gr** — `site:naftemporiki.gr [topic]`
6. **newmoney.gr** — `site:newmoney.gr startup [this week]`

### Money + rules (timing signals)
7. **espa.gr** — open EU calls. `ΕΣΠΑ ανοιχτές προσκλήσεις [category] [year]`
8. **gov.gr / minecon.gr** — `νόμος [category] 2026 ελλάδα`

### Truth (what real people say — competitor weakness = your opening)
9. **reddit r/greece** — `[category] site:reddit.com/r/greece`
10. **Google Maps / Trustpilot reviews** — `[competitor] reviews Greece` (read 1-star snippets)

### Anchors (verify a hard number when needed)
11. **INSETE** (tourism) — `INSETE [metric] 2026`
12. **ELSTAT / Bank of Greece** — `ΕΛΣΤΑΤ [metric]` (only when a fact needs a real source)

## DIASPORA (once a week, 1 query)
- `Greek diaspora [product] can't find abroad` OR `site:reddit.com/r/greek [topic]`

## HOW TO USE EACH RUN
1. Take your 3 research targets → write ONE sharp query per target (with `site:` if it fits a source above).
2. Add 1 query for the most overdue ACTIVE signal. That's ~4 searches total.
3. Read snippets. **If 3 snippets agree, stop — you have a STABLE_FACT.** Don't open pages to "read more."
4. Fetch a full page only if a number you must have appears in zero snippets.

The win isn't checking more sources. It's asking sharper questions of fewer.
